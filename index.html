  • nd Line Interface (CLI) - a text interface navigated by typing commands instead of using the mouse.
  • Terminal - Mac version of command line interface
  • pwd - print working directory
  • hostname - computer's network name
  • mkdir - make directory
  • touch - create an empty file
  • cd - change directory
  • ls - list directory
  • cp - copy a file or directory
  • cat - print the whole file
